Coding Expirence

The games above were mostly programmed in Unity using C#. The course project was written in ZGameEditor using its own scripting language.

XiheFramework

A game framework developed for agile development in Unity. Integrated common utility functions and editor tools such as Bezier curve, common modules like Event module, Blackboard module, etc. This project is still under heavy development.

SplitAnim

Split the rotation on an original bone evenly to target bones using constraints. Most humanoid retargeting tools consider the leg as thigh > shin > foot. Useful when the user wants something like thigh> knee > shin > foot or even thigh > knee > knee.001 > shin > foot.

MergeLoose

Merge the loose parts within an object by performing a boolean union operation. Useful to remove overlapped surfaces to increase performance.
